# Edge-AI Pipeline

The inference system performs:

1. Feature normalization
2. OneClassSVM inference
3. Mahalanobis distance analysis
4. Physics validation
5. Ensemble fusion scoring

All processing executes directly on ESP32 hardware.

# ML Training Pipeline

The ML workflow includes:

- Synthetic dataset generation
- Feature scaling
- OneClassSVM training
- Autoencoder training
- Mahalanobis model computation
- Ensemble fusion
- Export to ESP32 headers
