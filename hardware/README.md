# Hardware Folder

This folder contains the hardware setup, wiring information, and bill of materials for the motor fault detection system.

---

# Files

| File | Description |
|---|---|
| bom.csv | Bill of materials |
| hardware.jpeg | Hardware prototype image |
| wiring_connections.md | ESP32 wiring details |

---

# Hardware Components

| Component | Purpose |
|---|---|
| ESP32 | Main controller |
| ACS712 | Current sensing |
| MPU6050 | Vibration sensing |
| DS18B20 | Temperature sensing |
| IR Sensor | RPM sensing |
| L298N | Motor driver |

---

# System Features

- Multi-sensor monitoring
- Real-time acquisition
- Embedded Edge-AI inference
- Low-cost implementation
- Portable hardware architecture

# Sensor Fusion

The system combines:

- RPM
- Current
- Vibration
- Temperature

to compute a final anomaly score.
